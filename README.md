# CakeBliss 🎂

A simple and elegant cake shop landing page built with HTML and CSS.

## Live Demo

🌐 **Visit the live website:** [https://harshitha1106.github.io/cakeBliss](https://harshitha1106.github.io/cakeBliss)

## Features
- Hero section highlighting signature cakes
- Menu section with cake images
- Responsive layout for different screen sizes
